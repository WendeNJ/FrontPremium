# Premium Distribuidora - Frontend

## Instalação

```bash
npm install
npm run dev
```

## Estrutura

- src/pages/ - Páginas da aplicação
- src/components/ - Componentes reutilizáveis
- src/api/ - Cliente de API (configure conforme seu backend)

## Configuração

Edite src/api/base44Client.js e configure a URL do seu backend.

## Endpoints necessários

- GET/POST/PUT/DELETE /api/noticias
- GET/POST/PUT/DELETE /api/cards-mural
